FAQS

Have any question

What is Engram?

+

Engram is a local-first extension that creates a semantic index of your codebase, allowing for instant context retrieval and smart queries without data leaving your machine.

How does this help vibecoders?

+

Vibecoding is all about flow. Engram keeps you in the zone by automatically surfacing the right context and patterns, so you never have to break your rhythm to search for docs or implementation details.

What are the benefits of Engram?

+

It enhances productivity by establishing deep context awareness in your editor, reducing time spent searching for logic or references manually.

Is my code processed in the cloud?

+

No. We prioritize privacy. All embedding generation and vector search operations happen 100% locally on your device.

Which editors are supported?

+

Currently, Engram is optimized for Visual Studio Code, with support for JetBrains IDEs coming soon.

Is it free to use?

+

Yes, the core local extension is free to use for individual developers.
