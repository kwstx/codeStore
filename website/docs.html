Welcome to Engram information central. Here you'll find everything you need to set up your local coding companion.

Installation

Engram is distributed as a VS Code extension. To install:

  1. Open Visual Studio Code.
  2. Go to the Extensions view (Ctrl+Shift+X).
  3. Search for "Engram".
  4. Click Install.

First Run

After installing, open a folder or workspace. Engram will automatically prompt you to start indexing. This process runs locally and creates the semantic map needed for context retrieval.

Depending on your codebase size, this may take a few minutes. You can continue working while indexing happens in the background.

Features

  • Semantic Retrieval: Instantly find code logic based on meaning, not just exact keywords.
  • Ollama Integration: Connect to local LLMs to chat with your codebase without data leaving your machine.
  • Instant Context: Automatically surface relevant documentation and implementation details while you type.
